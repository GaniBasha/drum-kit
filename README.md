# Drum Kit 🥁

An interactive Drum Kit web application built using HTML, CSS, and JavaScript.

## Features

- Play drum sounds by clicking buttons
- Play drum sounds using keyboard keys
- Button animation effects
- Responsive and interactive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Keys and Sounds

| Key | Sound |
|-----|------|
| W | Tom 1 |
| A | Tom 2 |
| S | Tom 3 |
| D | Tom 4 |
| J | Crash |
| K | Kick Bass |
| L | Snare |
